3. Que es posicionamiento

Se entiende por posicionamiento al estilo que se le da para poder mover elementos y una vez el elemento está posicionado, aprende nuevas propiedades

4. Explicar los tipos de posicionamiento

Relative: No afecta nada, es decir, conserva su espacio

Soy el relative

Absolute: Necesita un contexto para moverse, es decir, el relative

Soy el absolute

Fixed: El contexto de fixe es toda la vista, no importa el padre o el ancestro

Soy el fixed

Sticky: El elemento es posicionado relativamente, hasta que llegue a un umbral específico

Soy el sticky

5. Explicar las propiedades que se aprende con el posicionamiento

Top: Es el posicionamiento en la parte superior

Right: Es el posicionamiento en la parte derecha

Bottom: Es el posicionamiento en la parte inferior

Left: Es el posicionamiento en la parte izquierda

Z-index: Es el posicionamiento de capas, mientras más alto sea el valor del z-index, se impondrá en la posición
